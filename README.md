# 🚀 Flutter Omni App (Практические работы 1-5)

Единое мультимодульное Flutter-приложение, объединяющее 5 практических заданий по разработке мобильных приложений. 

Проект построен с упором на **чистую архитектуру**, **плавные анимации** и **независимость от сети/локальных ассетов**. Все визуальные элементы генерируются математически с помощью встроенных инструментов Flutter.

---

## 📂 Структура модулей

### 🛠 Практика 1: Карточки, Меню и Анимации
- Кастомный `Drawer` (боковое меню) с бейджами уведомлений.
- Математический генератор градиентов и иконок.
- **Фишка:** Динамичная каскадная анимация появления элементов сетки (Scale + Fade + Slide с эффектом пружины `easeOutBack`).

### 🧮 Практика 2: Калькулятор депозита
- Защищенный парсинг данных (`double.tryParse`) для предотвращения крашей при пустом вводе.
- **Фишка:** Анимация "накрутки" итоговых чисел (счетчик от нуля до результата за 1.2 секунды).
- Автоматическое скрытие клавиатуры при расчетах.

### 🧠 Практика 3: Интерактивная викторина
- Использование пакета **Provider** для реактивного управления состоянием.
- Кастомный анимированный Splash Screen с эффектом пульсации.
- **Фишка:** Интерактивный UI — при выборе ответа кнопка плавно подсвечивается зеленым (успех) или красным (ошибка), давая задержку перед переходом к следующему вопросу.

### 🍵 Практика 4: Магазин чая "Лесной дом"
- Реализация многовкладочного каталога (`TabBar` + `TabBarView`).
- **Фишка:** Использование **Hero Animations**. При клике на товар из каталога, иконка плавно перелетает и масштабируется на экран деталей.
- Полностью рабочий счетчик цены (количество * базовую цену).

### 🍑 Практика 5: Фруктовый магазин
- Создание идеальных кнопок-капсул в `TabBar` с облегающим индикатором.
- **Фишка:** Использование системных эмодзи в качестве высококачественных векторных 3D-изображений товаров.
- Анимация бесконечного "дыхания" (ScaleTransition) товара на экране деталей.

---

## 🛠 Технологический стек
- **Framework:** Flutter (Dart)
- **State Management:** Provider (для Практики №3)
- **UI/UX:** Material 3, Custom Animations, Gradients

## 🚀 Как запустить проект

1. Убедитесь, что у вас установлен [Flutter SDK](https://docs.flutter.dev/get-started/install).
2. Склонируйте репозиторий на свой компьютер:
   ```bash
   git clone <https://github.com/EugeneWinter/FlutterOmniApp>
   ```
3. Перейдите в папку с проектом (замените `название_папки` на реальное имя папки):
   ```bash
   cd название_папки
   ```
4. Очистите кэш и подтяните все необходимые зависимости (включая пакет `provider`):
   ```bash
   flutter clean
   flutter pub get
   ```
5. Запустите приложение на эмуляторе, физическом устройстве или Windows Desktop:
   ```bash
   flutter run
   ```
